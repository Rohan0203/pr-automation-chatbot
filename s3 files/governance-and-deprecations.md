# Governance & Deprecations

## Deprecated Patterns

- Source buckets not aligned to Enterprise / Function
- Source-system-specific buckets (e.g. dev-lh1-sap-src)

Any deprecated pattern → **ERROR**

---

## Governance Principles

- Buckets are global AWS resources
- Dev/Prod collision avoidance is mandatory
- Ownership must be clear and auditable
- Provisioning applies governed automation to shared infrastructure
